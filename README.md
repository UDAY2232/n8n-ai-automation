# n8n-ai-automation
This project demonstrates an end-to-end AI-powered automation system built using n8n. It automatically fetches trending data from external APIs, processes and transforms it, generates intelligent content using OpenAI, and performs automated publishing actions such as uploading content via APIs.
# 🚀 AI-Powered Content Automation using n8n

## 📌 Overview
This project is an automated workflow built using n8n that:
- Fetches trending data from Google Trends
- Processes and transforms data (XML → JSON)
- Generates content using OpenAI LLM
- Automates API calls and publishing (YouTube integration)

## ⚙️ Features
- Scheduled automation (cron jobs)
- AI content generation
- Multi-step API workflow
- End-to-end automation pipeline

## 🛠 Tech Stack
- n8n
- OpenAI API
- REST APIs
- Docker

## 📂 Workflow
Import the JSON file into n8n to run this workflow.
